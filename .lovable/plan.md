

# 디지털 윷놀이 말판 (Digital Yutnori Board) - MVP

React + Tailwind CSS 기반의 디지털 윷놀이 보드 웹앱. 실물 윷은 오프라인에서 던지고, 앱은 말의 위치와 상태만 관리하는 디지털 보드 역할.

---

## 1단계: 팀 설정 화면 (시작 페이지)
- 두 팀(청팀 🔵, 홍팀 🔴)의 **팀 이름** 입력 필드
- 각 팀의 **말 개수** 설정 (기본 4개, 1~5개 범위)
- **"게임 시작"** 버튼 → 게임 보드 화면으로 이동
- 깔끔하고 직관적인 UI

## 2단계: 윷판 보드 구현
- **전통 윷판 레이아웃**: 5×5 정사각형 외곽 + 내부 대각선 X자 경로 = **총 29개 노드**
- SVG 또는 Canvas 기반으로 노드와 경로선을 시각적으로 렌더링
- 각 노드는 원형으로 표시, 경로는 선으로 연결
- **반응형 디자인**: 태블릿/TV 등 큰 화면에 최적화, 모바일에서도 사용 가능

## 3단계: 말 드래그 앤 드롭 이동
- 각 팀의 말을 **자유롭게 드래그 앤 드롭**으로 29개 노드 어디든 이동 가능 (이동 규칙 제약 없음)
- **자동 스냅(Snapping)**: 말을 노드 근처에 놓으면 해당 노드 중앙으로 자동 고정
- 부드러운 드래그 애니메이션

## 4단계: 말 업기(Stacking) 표시
- 같은 팀 말이 동일 노드에 2개 이상 위치하면, 말 하나에 **숫자 배지(×2, ×3...)** 표시
- 시각적으로 겹침 상태를 명확하게 표현

## 5단계: 게임 로그 & 상태 관리
- 화면 우측 또는 하단에 **이동 히스토리 로그** 표시 (예: "청팀 1번 말이 노드 A로 이동했습니다")
- 모든 게임 상태(말 위치, 팀 정보, 로그)를 **sessionStorage에 자동 저장** → 새로고침해도 유지
- **"게임 종료/초기화"** 버튼: 세션 초기화 후 팀 설정 화면으로 복귀
